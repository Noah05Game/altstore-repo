How to Update Your Repository
Step 1 — Add a new app or version
Use the Apps tab to add or edit an app. For updates, click the app name, then "Add Version" and fill in the new version details.
Step 2 — Upload your IPA
Place your .ipa file in the /apps/ folder of your GitHub repo. Name it com.bundle.id-1.2.3.ipa.
Step 3 — Upload your icon
Place your icon PNG in /icons/. Use a 512×512 PNG named com.bundle.id.png.
Step 4 — Export and replace apps.json
Click "Export JSON" above and replace the apps.json file in your repo root.
Step 5 — Commit and push
Commit all changes to your main branch. GitHub Pages auto-deploys in ~60 seconds.
Step 6 — Users get the update
AltStore and SideStore check for updates periodically. When the version number in apps.json is higher than the installed version, users see an update in the Updates tab.
💡 Tip: Never change the bundleIdentifier of an existing app — AltStore uses it to track installed apps. Only the version field inside versions[] triggers an update.
